Connector

A Connector operation represents a target or source of energy, information or matter of a spatial region.

Arguments

  • attribute: : String - Name of the attribute of the cells contained in the collections over which  the Flow will operate.
  • collection: : Cellular Space or Trajectory - A collection of cells that will be used to calculate  the Flow source or target.
  • neight: : Neighborhood - Neighborhood name defined on the collection of the energy Flow. Optional.
  • secundaryAttribute: : String - Name of the attribute of the cells contained in the collections  over which the Flow will operate. Optional. Atencion. If used, the flow rule must have an extra argument. Ex:  f (t, stock, stock2).

Usage


import("ssd")
---------------------------------------------------------------
-- # SPACE # Creation
fullCell = Cell {
   stock = 100
}
cs = CellularSpace {
   xdim = 9,
   instance = fullCell
}
-- ConnectorS
cs_localCnt = Connector {
   collection = cs,
   attribute = "stock"
}
