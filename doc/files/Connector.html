Connector

A Connector operation represents an target ou source of nergy, information or matter of a spatial region.

Arguments

  • attribute: : String - Name of the attribute of the cells contained in the collections over which the Flow will operate.
  • collection: : Cellular Space or Trajectory - A collection of cells that will be used to calculate the Flow source or target.
  • neight: : Neighborhood - Neighborhood name defined on the collection of the energy Flow. Optional.
  • secundaryAttribute: : String - Name of the attribute of the cells contained in the collections over which the Flow will operate. Optional.

Usage

import("ssd")
cell = Cell {
stock = 100
}
cell2 = Cell {
stock = 0
}
cs = CellularSpace {
xdim = 3,
instance = cell
}
cs2 = CellularSpace {
xdim = 3,
instance = cell2
}
timer = Timer {
Event {
action = function()
cs:synchronize()
cs2:synchronize()
return false
end
},
}
-- ConnectorS
cs_localCnt = Connector {
collection = cs,
attribute = "stock"
}
cs2_localCnt = Connector {
collection = cs2,
attribute = "stock"
}
-- Flow OPERATORS
vertical_local_Flow = Flow {
rule = function(t, stock) return 0.5 * stock end,
source = cs_localCnt,
target = cs2_localCnt,
timer = timer
}
-- MODEL EXECUTION
timer:run(1)
